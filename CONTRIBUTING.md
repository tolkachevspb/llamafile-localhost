# Contributing

Спасибо за интерес к проекту.

Этот репозиторий задуман как практичный локальный AI-стек для macOS Apple Silicon на базе `llamafile`. Поэтому лучшие вклады сюда — те, которые делают проект проще, стабильнее и понятнее для новых пользователей.

## Что особенно приветствуется

- исправления в документации
- улучшения bootstrap и helper-скриптов
- повышение воспроизводимости первого запуска
- улучшение UX для локального UI и OpenAI-compatible API
- дополнительные примеры интеграции с локальными инструментами и agent PoC
- более понятные troubleshooting-разделы

## Что важно сохранить

- минимум зависимостей
- отсутствие лишней инфраструктурной сложности
- явную ориентацию на macOS Apple Silicon
- простой путь `clone -> bootstrap -> start -> browser`

## Перед PR

Пожалуйста:

1. Проверьте, что изменение действительно помогает реальному локальному сценарию использования.
2. Не коммитьте модели, бинарники, логи, PID-файлы и локальный `config.env`.
3. Если меняете скрипты, убедитесь, что они остаются совместимыми с обычным `bash`.
4. Если меняете документацию, старайтесь писать коротко и практично.

## Локальная проверка

Полезный минимальный набор:

```bash
bash -n bootstrap.sh
bash -n start.sh
bash -n stop.sh
bash -n llm.sh
python3 -m py_compile client.py
```

Если у вас уже скачаны модели и рантайм:

```bash
./llm.sh status
./llm.sh health
./llm.sh test "Проверка после изменений"
```

## Стиль изменений

- предпочитайте небольшие, сфокусированные PR
- не усложняйте архитектуру без сильной причины
- если добавляете новый сценарий, объясните, зачем он нужен обычному пользователю
- если добавляете интеграцию, лучше приложить короткий рабочий пример

## Большие изменения

Для крупных изменений полезно сначала открыть issue и коротко описать:

- что именно вы хотите изменить
- зачем это нужно
- как это повлияет на текущий простой bootstrap-flow

Это поможет не увести проект в лишнюю сложность.
